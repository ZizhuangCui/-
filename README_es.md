# CommentGuard Killer

Panel local para supervisar comentarios de cuentas autorizadas.

El proyecto exporta reportes CSV, resalta coincidencias de palabras de riesgo y comentarios con alta interacción, y puede enviar resúmenes compactos mediante un bot de Feishu.

Úselo solo con cuentas propias o autorizadas y respete las reglas de cada plataforma, la privacidad y la legislación aplicable.

Consulte [README.md](README.md) para la guía actual de configuración.
