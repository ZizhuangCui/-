# 上海政府采购公告抓取工具

## 项目简介
自动抓取上海政府采购网公告信息，筛选弱电工程类项目，支持数据导出和报告生成。

## 文件说明

### 核心代码
- `fetch_sh_zfcg_v15_format.py` - 最新版本抓取脚本（推荐）
- `grab_shanghai_complete.py` - 完整版抓取脚本
- `send_report.py` - 报告发送脚本

### 可执行文件
- `grab_shanghai_final.exe` - Windows可执行版本

### 输出示例
- `上海采购公告_弱电工程类_完整版_v15_格式优化.xlsx` - 最新格式优化版数据
- `上海采购公告_弱电工程类_完整版_v14_优化.xlsx` - v14优化版数据

### 文档
- `招标开发文档.pdf` - 开发说明文档
- `弱电工程筛选报告.md` - 筛选报告说明
- `pdf_images/` - PDF文档中的截图素材

## 使用说明
1. Python版本需要安装依赖: requests, pandas, openpyxl
2. 运行 `python fetch_sh_zfcg_v15_format.py` 即可抓取最新公告
3. 输出文件为Excel格式，包含项目详情和链接

## 筛选规则
- 关键词匹配：弱电、智能化、安防、监控、网络、布线、机房、综合布线等
- 支持日期范围筛选
- 自动去重和格式优化
